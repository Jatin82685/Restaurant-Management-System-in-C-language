# Restaurant Management System (C)

This repository contains a console-based Restaurant Management System developed using the C programming language.

## Overview
The Restaurant Management System is a simple console application designed to manage basic restaurant operations.  
This project was created as part of my Computer Science studies at FAST University and focuses on strengthening core programming and problem-solving skills.

## Features
- Display restaurant menu
- Take customer orders
- Manage order details
- Calculate and generate total bill
- User-friendly console interaction

## Concepts Used
- Functions
- Conditional statements
- Loops
- Arrays
- Basic file handling (if applicable)

## Technologies Used
- C Programming Language
- Console / Terminal

## Usage
1. Compile the program using any C compiler (e.g., GCC).
2. Run the executable file in the terminal.
3. Follow on-screen instructions to place orders and generate bills.
